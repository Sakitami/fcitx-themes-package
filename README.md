# fcitx-themes-package

---

Fcitx themes package.

You can import these themes through the following command:

```bash
git clone https://github.com/Sakitami/fcitx-themes-package.git
cd fcitx-themes-package
chmod +x install.sh
sudo ./install.sh
```
